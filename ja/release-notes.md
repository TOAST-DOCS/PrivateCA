# リリースノート
**Management > Private CA > リリースノート**

## 2026. 03. 24.

### 機能追加
- API v2.0 ガイドにストアAPI(一覧照会、詳細照会、作成、修正、削除)を追加しました。
- API v2.0 ガイドに証明書(発行者)API(一覧照会、詳細照会、発行、失効)を追加しました。
- API v2.0 ガイドにテンプレートAPI(一覧照会、詳細照会、作成、修正、削除、証明書発行)を追加しました。
- ストア、証明書、テンプレートの一覧照会APIに、ページネーション及び検索クエリパラメータを追加しました。

### 機能改善・変更
- API v2.0のエンドポイントパスを `/cas/{caId}` から `/ca-stores/{caStoreId}` に変更しました。
- API v2.0 ガイドのリクエストフィールドに制約条件(最大長、値の範囲、デフォルト値など)を詳細に記載しました。

## 2026. 01. 27.

### 機能改善・変更
- バックデートの有効期間の最大値を30日に制限しました。
- **テンプレート > 有効期限設定**内の有効期限設定値の文言を証明書の有効期間に修正しました。
- 権限のないユーザーが操作した際のエラーメッセージを明確化しました。

### 不具合の修正
- テンプレート内の有効期限の指定方式を特定の日付に変更した際、TTLとして表示されるバグを修正しました。
- **ACME 詳細**タブで、特定の状況での画面遷移時にエラーモーダルが発生するバグを修正しました。

## 2025. 12. 23.

### 新規サービスリリース
- Private CAサービスを新規リリースしました。Private CAサービスを利用してRoot CA及びIntermediate CAを作成し、組織内部用の証明書を発行できます。ACMEプロトコルをサポートしており、Certbotなどのクライアントで証明書を自動的に発行及び更新できます。また、CRL(certificate revocation list)及びOCSP(online certificate status protocol)で証明書の失効状態を確認できます。
