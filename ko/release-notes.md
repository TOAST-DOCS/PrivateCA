# 릴리스 노트
**Management > Private CA > 릴리스 노트**

## 2026. 03. 24.

### 기능 추가
- API v2.0 가이드에 저장소 API(목록 조회, 상세 조회, 생성, 수정, 삭제)를 추가하였습니다.
- API v2.0 가이드에 인증서(발급자) API(목록 조회, 상세 조회, 발급, 폐기)를 추가하였습니다.
- API v2.0 가이드에 템플릿 API(목록 조회, 상세 조회, 생성, 수정, 삭제, 인증서 발급)를 추가하였습니다.
- 저장소, 인증서, 템플릿 목록 조회 API에 페이지네이션 및 검색 Query Parameters를 추가하였습니다.

### 기능 개선/변경
- API v2.0의 엔드포인트 경로를 `/cas/{caId}`에서 `/ca-stores/{caStoreId}`로 변경하였습니다.
- API v2.0 가이드의 요청 필드에 제약 조건(최대 길이, 값 범위, 기본값 등)을 상세히 기재하였습니다.

## 2026. 01. 27.

### 기능 개선/변경
- 백데이트 유효성의 최댓값을 30일로 제한하였습니다.
- 템플릿 > 만료 설정 내 만료 설정의 값을 인증서 수명으로 문구를 수정하였습니다.
- 권한이 없는 유저의 동작 시 오류 메시지를 명확하게 수정하였습니다.

### 버그 수정
- 템플릿 내 만료 방식을 특정 날짜로 수정 시 TTL로 노출되는 버그를 수정하였습니다.
- ACME 상세보기 탭에서 특정 상황의 화면 전환 간 에러 모달이 발생하는 버그를 수정하였습니다.

## 2025. 12. 23.

### 신규 서비스 출시
- Private CA 서비스를 새롭게 출시했습니다. Private CA 서비스를 이용해 Root CA 및 Intermediate CA를 생성하여 조직 내부용 인증서를 발급할 수 있습니다. ACME 프로토콜을 지원하여 Certbot 등의 클라이언트를 통해 인증서를 자동으로 발급 및 갱신할 수 있습니다. 또한 CRL(certificate revocation list) 및 OCSP(online certificate status protocol)를 통해 인증서 폐기 상태를 확인할 수 있습니다.
